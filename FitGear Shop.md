# FitGear Shop

**FitGear Shop** is a simple and clean front-end web project designed to showcase fitness equipment products. It focuses on presenting product details such as name, price, category, and description in an organized and user-friendly interface.  
This project is ideal for beginners learning how to structure a small web application using **HTML**, **CSS**, and **JavaScript**.
